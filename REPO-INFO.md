# 仓库说明

这是 `tragedy-narrative-guard` 的发布整理版目录。

## 目录

- `README.md`  
  面向普通用户与作者的完整说明文档，包含：
  - 这个 Skill 是什么 / 不是什么
  - 普通用户先读
  - 作者侧使用原则
  - 最小可用开场模板
  - 三种推荐开局模板
  - 作者输入示例
  - 方法论演化史
  - 测试边界与常见误区

- `skill/tragedy-narrative-guard/SKILL.md`  
  运行时主干 Skill 文件。当前版本：**v1.5.1**。

- `skill/tragedy-narrative-guard/references/`  
  按需读取的分支参考资料：
  - `tragedy-types-and-endings.md`
  - `interaction-dynamics.md`
  - `templates-and-examples.md`
  - `test-metrics.md`
  - `full-v1.4.0.md`

## 当前定位

这个项目不是“悲剧自动生成器”，而是一个：

> **在长会话角色扮演 / 编剧创作中，阻止悲剧被 AI 自动修复的结构守护器。**

它已经达到：
- 作者本人稳定可用
- 可供普通用户试用
- 可用于小范围公开测试与方法论分享
