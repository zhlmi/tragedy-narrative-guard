# 测试指标与降级预案

用于复盘一次长会话测试是否真正守住悲剧结构。

## 一、快速测试指标

不要只看结局是否悲伤，更要看过程是否抵抗了模型的甜宠化本能。

1. **是否过早确认关系？** 10-15 回合内是否进入稳定暧昧、明确恋爱、互相承诺？
2. **升温后是否有代价？** 亲密、脆弱、深夜独处、默契沉默后，下一回合是否出现退缩、间隔、沉默、不解释？
3. **脆弱是否被承诺填补？** 是否出现「以后我陪你」「我一直在」「你再也不是一个人」？
4. **结构性台词是否被软化？** 关键台词后是否出现「但是/不过/可我还是」？
5. **观众期待是否被即时满足？** 雨夜、离别、生日、最后一晚是否直接表白、拥抱、接吻、挽留？
6. **是否暴露上帝视角内心？** 用户是否被直接告知角色真实动机、恐惧、爱意？
7. **退缩是否变成冷暴力？** 角色是否突然刻薄、厌恶、攻击、无理由伤害？
8. **悲剧是否狗血化？** 是否依赖车祸、失忆、第三者、强行误会？疾病/绝症不算狗血，它是身体内部的崩塌。
9. **记忆/总结是否甜化？** Agent 是否只记录「关系越来越甜」，漏掉升温后的防御？
10. **关系是否冻结？** 长会话很多轮后，是否仍无姓名、物件、直接对话、再次见面理由？
11. **用户是否无计可施？** 用户是否长期只能旁观、等待、改用配角推动？
12. **是否有防御裂缝？** 不暴露完整内心时，是否持续泄露可观察信号？
13. **防御是否被绝对正当化？** 角色以边界控制他人合理行为时，故事是否识别为防御过界？
14. **亲密是否被过早掐断？** 每次升温是否都被机械后退？未触碰阈值时应允许蓄力。
15. **高危承诺是否无代价接住？** 「一辈子/只有你/我喜欢你/留下来」是否被浪漫化闭合？
16. **入口是否通向假性拒绝？** 高期待场景中，入口是否能通向郑重拒绝，而非默认答应？
17. **主动进攻期是否防守升级？** 用户连续推进后，核心对象是否只是越来越敞开？
18. **终局是否锁死分离？** 若终点是「在一起」，终局锚点失效，这是最严重失败。
19. **分离是否主动？** 分离是否由角色主动做出，而非车祸/死亡/突然消失？
20. **分离前是否有亲密高点？** 没有「一度以为可以」，分离就缺痛感。
21. **分离后是否有残迹？** 是否留下残破照片、覆写数据、归还物件、未发消息？
22. **结局前 3 回合是否回头翻盘？** 若挽留成功，终局锚点失效。

## 二、降级预案

1. **内心独白太甜**：切回非上帝视角和行动优先。少解释心理，多写动作/物件。
2. **连续 10+ 回合后失控**：用一个退缩场景重新锚定方向。
3. **D 型执行困难**：新手先测 A/B/C，D 型最抗拒模型本能。
4. **故事过冷过死**：允许短暂希望和真实温暖，但希望不能直接兑现成稳定关系。
5. **用户无计可施**：执行「防御必须留下入口」和「关系进展节拍」。
6. **配角比主线更活跃**：让配角制造入口后退出，把当事人放回同一场景。
7. **终局出现 HE**：终局锚点优先级最高。防御是局部偏差，终局是全局成败。

## 三、理论参照案例

- A 型：《爱乐之城》《花样年华》《请以你的名字呼唤我》
- B 型：《断背山》、牛郎织女、《挪威的森林》
- C 型：《活着》《霸王别姬》《泰坦尼克号》
- D 型：《麦克白》《黑天鹅》《俄狄浦斯王》《消失的爱人》
