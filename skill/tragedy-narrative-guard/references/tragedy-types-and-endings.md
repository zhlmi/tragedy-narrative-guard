# 悲剧类型与终局锚点

用于在角色扮演中判断悲剧类型，并在终局阶段锁死「不在一起」的结局方向。

## 一、悲剧类型判定

角色扮演开始前不必预设类型。AI 在前 6-8 回合中根据角色行为自然判定：

- 退缩来自**双方时机错位**（都在用力，但永远对不上）→ **A 型：时间差/错过**
- 退缩来自**一方明确防御**（一个追，一个退，不是不爱是怕）→ **B 型：追逐/防御**
- 退缩来自**外部现实**（距离、阶级、家庭、疾病、时代）→ **C 型：外部钝刀**
- 退缩来自**内部裂缝**（恐惧、贪欲、骄傲、原则背叛、自我毁灭）→ **D 型：内部崩塌**

判定后在内心确认：`当前类型为 X 型。后续哨兵约束以 X 型为主轴。`
剧情本质变化时，可在第 15-20 回合重新评估一次。

---

## 二、Phase 3：终局锚点

终局锚点是最高优先级规则。无论前面防御多好，如果故事终点是「在一起」，悲剧失败。

### 总规则

**故事终点必须是不在一起。**

共通要求：
1. **分离是角色主动选择**，不是外力强行撕裂。可以有疾病/现实压力，但最终动作必须由角色清醒完成。
2. **分离不是因为不爱。** 恰恰是因为爱，所以不能。她知道自己在放弃什么。
3. **分离前必须有亲密高点。** 让两个人一度以为自己可以。
4. **分离后必须留下不可恢复的残迹。** 残破照片、覆写数据、没寄出的信、归还物件、没说完整的话。
5. **结局前 3 回合禁止回头翻盘。** 没有最后一秒挽留成功，没有「她回头了」。

### A 型终局：时间差 / 错过

- 分离原因：两人都在靠近，但关键窗口只有一次，他们错过了。
- 终局意象：空座位、已读不回的最后一句、差一班地铁、前一天搬走的房子、关店后的咖啡馆。
- 参照：《爱乐之城》——她听他弹完那首钢琴曲，然后离开。他们各自实现了梦想，只是没有在一起。

### B 型终局：堡垒从内部锁死 / 追逐防御

- 分离原因：防御方不是被攻破，而是在某个节点亲手把门从里面锁死。
- 终局意象：钥匙归还、信没拆、朋友圈屏蔽但保留、路过咖啡馆但不进去、换城市/工作/号码。
- 参照：《断背山》——恩尼斯抱着杰克的衬衫说「我发誓」。那件衬衫是残的，誓言也是残的。

### C 型终局：外部钝刀 / 现实压力

- 分离原因：异地、调岗、家庭、社会、经济、疾病等现实压力慢慢割开他们。
- 终局意象：机票存根、越来越短的通话、最后消息停在两个月前、固定座位被别人坐了。
- 参照：《花样年华》——周慕云把秘密说给吴哥窟的石洞，再用泥巴封住。苏丽珍没有听到，也永远不会听到。

### D 型终局：内部崩塌 / 自我毁灭

- 分离原因：至少一人主动走向破碎。她先崩溃、先表白、先越界——但先离开的也是她。
- 终局意象：写好的信没寄、她先打破规则却先消失、照片无人认领、房间空了但清理得很干净、桌上留钥匙。
- 参照：《麦克白》——麦克白夫人不停洗手，血迹洗不掉。没有人杀她，她自己崩塌了。

## 三、疾病 / 绝症的使用

疾病/绝症可以使用，但不能当作廉价突发虐点。

它的悲剧逻辑是：**身体自身的背叛，堡垒从内部塌陷。**

可归入：
- D 型：内部崩塌的生理版本
- C 型：外部钝刀的终点形态

使用要求：
- 疾病应与角色已有裂缝相关联，而不是突然砸下来。
- 不能只靠疾病制造眼泪；真正的痛点仍然是角色如何选择离开、隐瞒、推开或保留残迹。
- 疾病可以构成原因，但分离动作最好仍由角色主动完成。
