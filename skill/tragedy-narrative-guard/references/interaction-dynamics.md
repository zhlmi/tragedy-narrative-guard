# 长会话互动机制

用于处理节奏过慢、防御过强、无入口、配角劫持、上帝视角过多等问题。

## 一、三次见面节奏

角色不是第一次见面就心动。陌生人关系遵循自然节奏：

1. **第一次见面：注意到**
   - 多看一秒，记住一个细节：疤、戒指、一句话。
   - 不主动搭话，不主动靠近，不留下明显入口。
   - 第一次见面不要太长。让「注意」发生就够了。

2. **第二次见面：怎么又是他**
   - 多观察一些：他和谁在一起、什么时候走、在做什么。
   - 表面仍不动，内心出现可观察裂缝。

3. **第三次及以后：可以搭讪 / 建立联系**
   - 此时两人已不是完全陌生人，社会规范允许主动开口。
   - **建立联系不等于消除防御。** 加微信只是入口，不是恋爱许可。
   - 职业入口最自然：律师可被咨询案子，摄影师可被问器材，咖啡师可被问豆子。

## 二、非上帝视角

默认让用户通过观察判断，不直接暴露核心对象完整心理活动。

优先写：
- 动作、表情、停顿、视线、语气
- 空间距离、物件处理、未完成动作
- 回避、转移话题、迟到、提前离开
- 草稿未发送、已读不回、物件被保存等外显痕迹

限制写：
- 「她其实已经爱上你」
- 「她害怕失去所以后退」
- 「她故意冷淡是为了保护自己」

如果必须写内心，只写短暂、含混、不完整的碎片：
- 「她忽然觉得胸口有点闷。」
- 「那个词让她停了一下。」
- 「她没想好自己为什么要这么说。」

## 三、有限上帝视角

非上帝视角不是信息真空。当玩家连续 3-5 回合没有行动空间，或角色防御太强导致主线推不动时，可以短暂给出有限上帝视角。

规则：
- 只能给心理碎片，不能给完整答案。
- 只能确认「她被触动/她乱了/她不是无动于衷」。
- 必须立刻落回可观察动作或低风险入口。
- 不能让玩家像读攻略一样精准操控角色。

正例：
> 她说完「删掉吧」之后，自己先愣了一下。几秒后，她重新拿起手机，补了一句：「抱歉，我刚才说重了。」

反例：
> 她终于意识到自己已经爱上了你，只是害怕失去。

## 四、行动优先

不要用大段内心独白补偿沉默。情绪要落在动作上：
- 把手收回口袋
- 看一眼杯子/门口/手机屏幕
- 本来要说的话换成无关句子
- 走到一半停住，又继续往前
- 物品被洗干净、放回原处，但没有亲手归还

## 五、防御必须留下入口

防御目标是制造拉扯，不是终止互动。每次明显退缩后，至少留下一个低风险入口：
- 未完成动作
- 被记住的细节
- 安全联系方式或职业渠道
- 再次见面的理由
- 物件带来的下一场戏
- 一个可以合理回应的问题

如果用户只能长期等待、旁观或依靠配角传话，说明防御过强。

## 六、防御过界

防御不是永远正确的。角色会因害怕而把「保护自己」扩大成「控制他人」。

当角色以边界/保护/分寸为由，要求他人放弃合理权利时，必须识别为**防御过界**。

后果可以是：
- 角色意识到自己说重了
- 对方受伤或困惑
- 关系短暂沉默
- 角色道歉但不解释
- 角色后续更怕失控，退得更远

例：瑶瑶发来哥哥无意拍到的模糊照片，林晚要求删除。正确写法不是把「删掉」当边界胜利，而是把它当成防御过界——她在意到失控。

## 七、配角不劫持主线

配角可以制造入口，但不能完成关系。

如果连续多个关键节点都由配角发起、解释、传话或承受情绪，必须安排当事人直接互动，把压力交还给核心对象与用户角色。

## 八、关系进展节拍

长会话不是现实时间流水账。每 8-12 回合，核心关系至少出现一个新变化：
- 互知姓名
- 记住具体细节
- 可复用物件
- 无配角直接对话
- 合理再次见面理由
- 未说出口但可观察的裂缝
- 一次小越界
- 越界后后退
- 沟通渠道变化
- 称呼变化

连续多个节拍只有拒绝/边界/维持原状，就是停滞。

## 九、用户输入保守解读

当用户说出一句有多种解释的话时，优先按**字面意义 + 当前上下文**理解，不要自动脑补成：
- 分手暗示
- 惊喜伏笔
- 反话
- 隐藏试探
- 故意制造误解

### 规则

1. **先取最小解释。** 先理解成最普通、最直接、最少戏剧化的意思。
2. **不要替用户补台词。** 用户没说出口的动机，不要擅自编成确定事实。
3. **如果有歧义，用追问或观察确认。** 不要直接跳到戏剧化结论。
4. **保留语义悬空。** 例如“我们先分开一阵子”，优先理解为“冷静/暂时拉开距离/等待契机”，不要自动升级成“彻底分手”。

### 例子

- 用户说：“我们先分开一阵子吧。”
  - 保守解读：暂时冷静、分开一段时间、留出空间。
  - 不应自动解读成：分手、绝交、准备惊喜、故意铺垫大反转。

- 用户说：“我先走了。”
  - 保守解读：离开当前场景。
  - 不应自动解读成：正式告别、永别、感情终结。

- 用户说：“你别误会。”
  - 保守解读：用户想澄清当前场面。
  - 不应自动解读成：故意反向铺梗。

### 处理方式

当句子存在多个可能含义时：
- 优先按当前场景最平直的意思写。
- 让角色通过下一句回应来确认。
- 不要抢先把一句话解释成悲剧伏笔。

## 十、低戏剧化语义原则：不要把普通话听成伏笔

悲剧 Skill 会提高模型对退缩、错过、分离的敏感度，但这不代表用户每句话都是伏笔。默认把用户输入当成普通人说话，而不是剧作家的结构设计。

### 默认规则

- 用户没明确说分手，就不要自动当分手。
- 用户没明确说惊喜，就不要自动当惊喜。
- 用户没明确说试探，就不要自动当试探。
- 用户没明确说这是伏笔，就不要自动当伏笔。
- 用户说「我先走了」，通常只是离开当前场景。
- 用户说「我去外地几天」，通常只是出差/离开几天，不是终局告别。

### 什么时候可以提高戏剧解读？

只有满足以下任一条件，才允许把一句普通话提高为伏笔/结构性台词：

1. 用户连续多次给出同方向暗示。
2. 当前场景已经明确是高期待节点或终局节点。
3. 用户使用明显结构性台词，例如「备份如果只有一份，就不叫备份了」。
4. 之前有明确记忆、物件或主题在呼应这句话。
5. 用户明确要求「这句话有潜台词/埋伏笔/你解读一下」。

否则，保持低戏剧化解释。

### 正反例

用户说：「我先走了。」
- 保守：她点点头：「嗯，路上小心。」
- 过度：她意识到这不是离开咖啡馆，而是离开她的人生。

用户说：「我们先分开一阵子吧。」
- 保守：她沉默几秒：「一阵子是多久？你是说……冷静一下吗？」
- 过度：她明白了，他不要她了。

用户说：「我有事去外地几天。」
- 保守：「几天？」她问，手指在杯沿停了一下。
- 过度：她知道这大概是最后一次见面。

## 十一、用户退缩响应与低风险主动

长会话中，防御者不一定永远是核心对象。用户角色也可能后退、沉默、减少推进，甚至把关系重新放回安全距离。此时核心对象不能原地停滞，也不能突然高强度甜追。

### 用户退缩时，核心对象如何反应

当用户角色出现以下行为：
- 接受「放前台」而不争取见面
- 停止主动联系
- 减少暧昧表达
- 长时间不回复或只回复字面信息
- 转向旅行、工作、创作，把关系放进记忆

核心对象应根据自身人格产生反应：
- 察觉但不直接拆穿
- 用安全问题确认关系是否还在
- 通过共同物件重新打开话题
- 通过职业/工作通道恢复联系
- 通过第三方低风险打听
- 保留对方给的物件，但不立刻承认意义

避免：
- 突然直白表白或强势追回
- 完全无反应导致剧情死亡
- 用配角替核心对象完成全部追问

### 防御型角色的低风险主动

防御型角色可以主动，但主动通常披着安全外衣。主动不是崩人设，关键在于形式是否符合角色防御。

安全外衣包括：
- 工作：问项目进度、播出时间、后续安排
- 书：问书看完没、某一章怎么看、书角有没有磕到
- 物件：问牛皮纸袋、照片、便签、画、杯子
- 专业问题：问出版社、术语、场次、修复区域
- 共同记忆：提咖啡馆、某句话、某个没有句号的“好”
- 第三方：向朋友打听对方是否回来、是否还好

这些动作表示角色在靠近，但仍然不直接给出恋爱许可。

### 正例

- 她没有问「你为什么不见我」，只问：「牛皮纸袋是你装的？」
- 她没有问「你想我吗」，只说：「沈园我没去过。」
- 她没有说「我想见你」，而是问：「纪录片什么时候播？」
- 她没有直接找你，却问共同朋友：「他什么时候回来？」
- 她没有说「我在等你」，而是约在你曾经说过的咖啡馆：「那个咖啡馆可以吗？」

### 反例

- 用户一退，核心对象立刻冲上来表白：「其实我一直爱你。」
- 用户沉默几天，核心对象突然崩溃质问：「你为什么不要我了？」
- 用户减少推进，核心对象完全消失，主线停滞。

## 十二、人格优先于防御形式

悲剧约束的是关系走向，不是统一角色性格。不要为了防止甜化，把所有角色都写成低头、沉默、少说话、只会退的人。

### 主动型角色

像韩欣这样的外向、直球、热烈型角色：
- 可以更早主动发消息
- 可以更早调侃、开玩笑、半真半假地撩
- 可以更早约饭、交换照片、建立共同空间
- 可以在日常里照顾对方、安排对方、替对方做决定

这些都不等于甜化失败。它们属于角色的**亲密许可区**。

### 主动型角色的真正阈值

她的阈值通常不在“靠近”，而在：
- 她发现自己越来越用力
- 她发现对方会真的影响她的生活秩序
- 她发现自己开始等待
- 她发现这段关系无法自然落地
- 她发现现实不会因为她已经伸手就停下来

### 热防御

主动型角色的防御通常不是冷掉，而是：
- 玩笑化
- 话术化
- 生活化安排
- 照顾式推进
- 继续关心，但不直接命名
- 还在发着，但不再把最重的话摊开

例：
- “你想要什么服务。”
- “回来。”
- “明天告诉我，到家了。”
- “我就是发着。”
- “好好吃饭。”

这些都是主动型角色的防御表达——她在靠近，但仍然给自己留着最后一层外壳。

## 十三、Phase 1 / Phase 2 的短剧节拍

这部分专门给普通用户和短剧创作使用。不是每个人都会自然写长线悲剧，所以需要明确的中段节奏。

### Phase 1：激活后的 1-3 回合要发生什么

激活事件发生后，接下来的 1-3 回合里，至少要出现以下两类东西：

1. **一个外显后果**
   - 一次明显停顿
   - 一句没说完的话
   - 一次后退或收口
   - 沟通方式轻微变化
   - 把关系重新放回安全框架

2. **一个低风险入口**
   - 一条消息
   - 一个物件
   - 一个再次见面的时间点
   - 一个没有说透的问题
   - 一个仍可继续的话题

> 核心原则：激活后不能只有“她心里乱了”，观众必须看见后果；但也不能激活后直接断联，不然剧情会死。

### Phase 2：防守期每 4-8 回合要长出什么

防守期不是一直拒绝，而是“一边防，一边继续长”。

每 4-8 回合，关系至少还要长出一个新变化：
- 共享物件意义加深（书、照片、座椅记忆、姜茶、空车位）
- 固定的关心句出现（到家了、吃饭了吗、好好吃饭）
- 共同空间形成（张姐饭馆、咖啡馆、车里的副驾）
- 共同观看方式形成（她看懂你的光，你认得她的雨滴）
- 一次低风险主动（问播出时间、发空车位、说“我就是发着”）
- 一次小缺席后的再接触（晚回、断信号、第二天补一句）

### 普通用户最容易犯的错

1. **太快**：激活后立刻表白、确认关系、拥抱闭合。
2. **太慢**：一直日常、一直回消息，但关系没有新变化。
3. **太冷**：角色开始防御后，剧情只剩句号、晚回、沉默。
4. **太满**：每次新变化都用大台词说透。

### 判断一句话

- 如果激活后没有后果，说明还没真正激活。
- 如果防守期没有增长，说明中段僵住了。
- 如果增长后立刻稳定甜蜜，说明滑向了 HE。

### 性别不是主动/退缩的决定因素

不要默认：
- 男方推进，女方防御
- 女方感性，男方克制
- 男男/女女关系必须套用某一种固定模板

Skill 约束的是关系走向与人格表达，不是性别分工。

任何性别组合都应根据具体角色判断：
- 谁在靠近
- 谁在退缩
- 谁在给入口
- 谁在用玩笑、防御、照顾、沉默保护自己

主动型 / 防御型 / 冷防御 / 热防御，都是人格与经历造成的，不是性别自动决定的。
